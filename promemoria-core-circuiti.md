# 🔥 Core — Protocollo Circuiti

> Promemoria della Scheda 2 — Focus alto (rev. 2026-06-08, v430).
> I 3 circuiti core distinti, uno per giornata.

## Regola comune a tutti e 3
- **2 giri** completi → sali a **3 giri** quando regge
- **Rec 1'** tra un giro e l'altro (dentro il giro: minimo, si scorre)
- **Progressione per QUALITÀ**, mai aggiungendo reps → ordine: rallenta il tempo → aggiungi il 3° giro → allunga il plank → zavorra
- Mappa: circuito più duro sul giorno più fresco → **A → Circ.3 · B → Circ.2 · C → Circ.1**

---

## 🔴 Circ.3 → giorno A (il più tosto, sul giorno fresco)
| Mossa | Dose |
|---|---|
| Crunch inverso | **10** |
| Hollow hold | **20-25"** (scala: ginocchia piegate se cedi) |
| Plank tocco spalla | **8 / lato** |

---

## Note tecniche
- **Dead bug / Bird-dog**: lombare incollata a terra (dead bug) o bacino fermo (bird-dog) — movimento lento e controllato, niente slancio.
- **Hollow hold**: se la lombare si stacca da terra, piega le ginocchia. Meglio 15" puliti che 25" inarcato.
- **Russian twist**: tenuto su tua scelta (alternativa più sicura per la lombare = side-plank reach-through) — busto controllato, non a strappi.
